# Aprendendo JavaScript

Bem-vindo ao repositório **aprendendo**! Este repositório foi criado com o objetivo de explorar e aprender conceitos fundamentais e avançados de JavaScript. Cada seção contém exemplos práticos e exercícios para ajudar a consolidar o aprendizado.

## Estrutura

O repositório está organizado em pastas, cada uma representando um tópico de estudo:

- **02_operadores**: Introdução e exemplos de precedência de operadores.
- **04_laçosDeRepeticao**: Laços de repetição como `while`, com exemplos práticos.
- **05_funcoes**: Demonstrações de funções, incluindo funções de ordem superior e callbacks.
- **06_arrays**: Trabalho com arrays, explorando suas operações e métodos.
- **07_objetos**: Manipulações e exemplos de uso de objetos em JavaScript.

## Recursos

Este repositório fornece exemplos e documentos para ajudar em seu aprendizado de JavaScript. Abaixo está o que você encontrará:

- **Funções**:
  - Declaração e chamadas básicas.
  - Funções anônimas e arrow functions.
  - Funções importadas entre arquivos utilizando `require`.

- **Objetos**:
  - Criação de objetos.
  - Métodos de iteração e manipulação de dados dentro de objetos.
  - Objetos aninhados e exemplos de vida real.

- **Arrays**:
  - Manipulação básica de arrays.
  - Métodos e funções de iteração.

- **Laços de Repetição**:
  - Aplicações práticas de `while` e `do-while` com cenários do dia a dia.

- **Exercícios**:
  - Desafios interativos para consolidar cada uma das áreas acima. Incluem quizes e problemas resolvidos diretamente no código.

## Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/CaueGomes07/aprendendo.git
   ```
2. Acesse a pasta do projeto no terminal:
   ```bash
   cd aprendendo
   ```
3. Explore os arquivos e pastas para estudar os exemplos e desafios.

## Tecnologias Utilizadas

- **Node.js**: Para executar exemplos interativos.
- **JavaScript**: Linguagem principal para demonstração dos conceitos.
